Support

If you need help with AI Product Importer Pro, this page explains where to report issues and what details to include so problems can be solved faster.

How to get help

For support, please open a GitHub issue in the public repository for this website or the main plugin repository, depending on where support is being managed.

Open website issues

Open plugin issues

Include these details

  • Your WordPress version
  • Your WooCommerce version
  • The plugin version you are using
  • Whether you are using a managed account or your own API key
  • The exact action that failed
  • Any visible error message
  • Relevant logs or screenshots, with secrets removed

Before you contact support

  1. Confirm the plugin is updated to the latest available build.
  2. Try the action again after refreshing the connection from the plugin settings if you are using the managed mode.
  3. Check whether the issue is specific to one upload, one product, or all requests.
  4. Remove any private tokens, billing details, or personal customer data before sharing screenshots or logs.

Response expectations

Support is provided on a reasonable-efforts basis. No guaranteed response time or service-level commitment is offered unless stated separately in a paid support agreement.

This page is set up for an early GitHub-based support workflow. Once you have a branded email address or help desk, replace the GitHub issue links with your final support channel.
